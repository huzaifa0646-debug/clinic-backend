                                  **************Clinic Management System (Backend)********************

A secure, robust Spring Boot backend API designed to streamline clinic operations, including patient data management, appointment scheduling, and secure user account recovery.

**Features**
Patient Management: CRUD operations for patient records.

Appointment Scheduling: Efficient booking system for clinic visits.

**Secure Authentication**: Role-based access control.

Password Recovery: Secure "Forgot Password" implementation using token-based recovery.

**Tech Stack**
**Language**: Java 26

**Framework**: Spring Boot 4.1

**Database:** PostgreSQL

**Security**: Spring Security
